## 🚀 How To Run Locally

**Step 1 — Clone the repository:**
```bash
git clone https://github.com/rugematotti-oss/rugema-s-portfolio.git
```

**Step 2 — Navigate into the folder:**
```bash
cd rugema-s-portfolio
```

**Step 3 — Open the site:**

Simply open `index.html` in your browser:
- On Windows: double click `index.html`
- On Mac: right click → Open with → Chrome/Safari
- With VS Code: install Live Server extension and click **Go Live**

No installation or dependencies required — it's pure HTML, CSS and JavaScript!
